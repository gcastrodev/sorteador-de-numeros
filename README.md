# 🎲 Sorteador de Números

Aplicação web simples desenvolvida com **HTML, CSS e JavaScript puro** que permite gerar números aleatórios dentro de um intervalo definido pelo usuário.

O usuário informa um **valor mínimo** e um **valor máximo**, e o sistema retorna um número aleatório dentro desse intervalo.

---

# 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

# 🧠 Conceitos aplicados

Este projeto demonstra conhecimentos em:

* Manipulação do **DOM**
* Captura de **inputs do usuário**
* **Funções** em JavaScript
* **Estruturas condicionais**
* Geração de números aleatórios com `Math.random()`
* **Validação de dados**
* **Responsividade com CSS moderno**

---

# 📱 Responsividade

O layout foi desenvolvido utilizando técnicas modernas de CSS como:

* **Flexbox**
* **Media Queries**
* **clamp() para tipografia e dimensões adaptáveis**

A responsividade hoje é considerada o **mínimo necessário no desenvolvimento web moderno**, já que a maior parte do tráfego da internet vem de dispositivos móveis.

Sites que não se adaptam a diferentes tamanhos de tela:

* perdem usuários
* prejudicam a experiência
* têm pior posicionamento no Google (mobile-first indexing)

Por isso o projeto foi desenvolvido pensando em **boa experiência tanto em mobile quanto em desktop**.

---

# 🎯 Funcionalidades

* Definir **número mínimo**
* Definir **número máximo**
* Gerar número aleatório dentro do intervalo
* Validação para impedir que o valor mínimo seja maior que o máximo
